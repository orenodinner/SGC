# 01 Product Charter

## プロダクト名
仮称: Lazy Gantt Desktop

## 一言で何を作るか
**入力が雑でも使える、Windows向けの軽量ガントチャート / タスク管理デスクトップアプリ**を作る。

## 誰のためのプロダクトか
- 複数の大きなプロジェクトを抱える個人
- 小規模チームの実務担当
- Excel 工程表文化から抜け切れない現場
- 毎日きっちり運用するのが苦手な人

## どんな痛みを解決するか
1. タスクを入れるのが面倒で、最初の一歩が重い
2. 日程変更が面倒で、現実と計画がすぐ乖離する
3. 大きい案件の内訳を追うと全体像が見えなくなる
4. 複数案件を横断すると、次の危険箇所が分からない
5. 年間計画を見ようとすると日次ツールでは窮屈
6. 最後は Excel 共有が必要になる

## プロダクト原則
- 最小入力で始められる
- 計画の修正が入力より速い
- 今日の運用と長期計画を同居させる
- 1画面で迷わせない
- Excel を敵にしない
- 「高機能」より「毎日開ける」を優先する

## 成功指標
- 初回起動から 5 分以内に、1つのプロジェクトと3つのタスクを作れる
- 期限変更がドラッグ操作 1 回で終わる
- 3 つ以上の大規模プロジェクトの次マイルストーンを 30 秒以内に把握できる
- 年間 / FY 表示で、四半期ごとの大きな予定が一覧できる
- Excel 出力と再取込で、主要列が欠損せず round-trip できる

## MVP でやること
- Quick Capture
- Inbox / Today / Overdue
- Project + WBS + Gantt
- Portfolio view
- Year / FY roadmap view
- Excel export/import
- Auto save
- Local backup
- 基本フィルタ / 検索
- 繰り返しテンプレート

## MVP でやらないこと
- 複雑な原価管理
- 多人数リアルタイム共同編集
- 高度なリソース平準化
- Chat / Wiki / Git / CRM の全部入り
- 高度な権限制御
- ガント依存関係の全種別対応

## 主要差別化要素
- 1行雑入力
- WBS 展開 + ポートフォリオ横断
- 年間 / FY 表示
- Excel round-trip
- Windows ローカルファースト
